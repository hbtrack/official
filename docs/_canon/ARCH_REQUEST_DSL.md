# ARCH_REQUEST DSL

ARCH_REQUEST é uma linguagem formal, não texto livre.

## Estrutura mínima
- Header determinístico
- Contexto
- Objetivos (MUST)
- SSOT
- Delta estrutural
- Gates

## Propriedades
- Determinístico
- Auditável
- Reprodutível

## Generation Rules

Rules for generating ARCH_REQUESTs are defined in:
→ `ARCH_REQUEST_GENERATION_PROTOCOL.md`

This DSL defines WHAT an ARCH_REQUEST is.  
The protocol defines HOW to generate one.
