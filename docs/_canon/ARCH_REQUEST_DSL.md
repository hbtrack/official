# ARCH_REQUEST DSL

ARCH_REQUEST é uma linguagem formal, não texto livre.

## Estrutura mínima
- Header determinístico
- Contexto
- Objetivos (MUST)
- SSOT
- Delta estrutural
- Gates

## Propriedades
- Determinístico
- Auditável
- Reprodutível

## Regras de Geração

As regras para gerar ARCH_REQUESTs estão definidas em:
→ `ARCH_REQUEST_GENERATION_PROTOCOL.md`

Esta DSL define O QUE é um ARCH_REQUEST.  
O protocolo define COMO gerar um.
