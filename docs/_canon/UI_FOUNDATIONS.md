# UI_FOUNDATIONS.md

## Objetivo
Definir fundamentos transversais de interface.

## Princípios
- clareza
- consistência
- acessibilidade
- responsividade
- previsibilidade
- feedback imediato

## Fundamentos
- grid
- spacing
- tipografia
- densidade
- contraste
- ícones
- estados de foco
- navegação por teclado

## Estados Globais de UI
- loading
- empty
- success
- error
- disabled
- offline
- partial-data

## Responsividade
- breakpoints: mobile-first (360, 768, 1024, 1440)
- targets primários: mobile e desktop (navegadores modernos)

## Acessibilidade
- contraste mínimo: WCAG 2.1 AA
- semântica e labels: todo input deve ter label visível ou `aria-label`
- foco visível: obrigatório
